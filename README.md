# Hanekokoro Fonts

Personal Iosevka builds.

* `Hanekokoro Mono`: Base font variant.
* `Hanekokoro Mono-E`: A variant of Hanekokoro Mono with only extended font.
* `Hanekokoro Sans`: Modified Iosevka Aile.
